Kết quả trả về trường hợp chạy "NO" (ảnh Result of 6_1).

- How to customize the options to users, e.g. only two options: “Yes” and “No”, OR “I do” and “I don’t” (Suggestion: Use Javadocs or using Eclipse/Netbean IDE help). 

Có thể tự tạo nút bằng cách dùng:
	JOptionPane.showOptionDialog()

Thay các lựa chọn ở trong Object[] options.

        Object[] options = {"I do", "I don't"};
        int choice = JOptionPane.showOptionDialog(
                null, //parent component
                "Do you want to change to the first class ticket?", //message
                "Custom Option", //title
                JOptionPane.DEFAULT_OPTION,
                JOptionPane.INFORMATION_MESSAGE, //type of mesage
                null, //icon
                options, //array of options
                options[0] //default option
        );

Để xử lý kết quả sau khi chọn, ứng với vị trí của option trong mảng options, trả về kết quả tương ứng:

if (choice == 0) {
    System.out.println("User chose I do");
} else if (choice == 1) {
    System.out.println("User chose I don't");
} else {
    System.out.println("User closed the dialog");
}

Ghi chú: Các câu hỏi được trả lời dựa trên kết quả chạy được bằng IJ IDE.
