If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?

- Câu trả lời là Java không cho phép.
Vì cả hai constructor đều có cùng kiểu tham số (String), Java không thể phân biệt được giữa chúng. Điều này dẫn đến lỗi trùng constructor, vì cơ chế nạp chồng (overloading) yêu cầu các phương thức phải khác nhau về số lượng, kiểu hoặc thứ tự tham số.
- Theo như câu hỏi:
DigitalVideoDisc(String title)
DigitalVideoDisc(String category)
-> giống hệt nhau về signature -> xảy ra lỗi.
